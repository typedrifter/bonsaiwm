# Roadmap

BonsaiWM is a work in progress. Here's the high-level direction:

- [ ] **Rename project**: rename files, variables ect. to reflect the for name
- [ ] **Meson build system**: modern build setup for subprojects and multiple source files
- [ ] **Lua integration**: runtime scripting via `~/.config/dwl/init.lua`
- [ ] **Gaps**: inner/outer gaps with smart gap support
- [ ] **SceneFX**: for rounded corners and effects
- [ ] **Rounded corners**: configurable border radius
- [ ] **Animations**: open/close slide+fade effects

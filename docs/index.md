## What is BonsaiWM?

BonsaiWM is a compact, opinionated [Wayland] compositor based on [dwl]. It works out of the box with sane defaults and minimal dependencies. Configuration lives in a single `config.lua` file.

- **Opinionated defaults** that you can fully override
- **Minimal dependencies** beyond wlroots
- **Lua-configured**, no recompilation needed

[dwl]: https://codeberg.org/dwl/dwl
[Wayland]: https://wayland.freedesktop.org/
